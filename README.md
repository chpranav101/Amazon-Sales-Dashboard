# Amazon Sales Dashboard - Power BI

## Project Overview
Developed an interactive Amazon Sales Analytics Dashboard using Power BI to analyze:
- Sales
- Profit
- Shipping Cost
- Quantity Sold

## Features
- KPI Cards
- Country-wise Sales Analysis
- Market Analysis
- Category Analysis
- Shipping Mode Analysis
- Interactive Filters and Slicers

## Tools Used
- Power BI
- Excel
- Data Visualization
- Dashboarding

## Insights
- Identified top-performing countries and cities
- Analyzed sales distribution across markets
- Compared category-wise performance
